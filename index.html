Q & A

The main diffrences between web1,web2 and web3

* The web 1 was considerd as a "read only web", it allowed us to search for Information and read it.
  There was very little user interaction.

* The web 2 in contrast to his its ancestor is more interactive and collaborative for the users,
  they could contribute to the content of the web (blogs, wikipedia and more)

* The web 3 is known as the Semntic Web, and the main idea is data integration.
  By using metadata, "display only" data is converted to meaningful information,
  which can be located, evaluated, and delivered by software agents.


"Medicines side effects" Use Case

* Web 1 - The user inserts a known url ,that contains information about medicines side effects, to the
  browser.

* Web 2 - The user inserts a query "medicines side effects" to a search engine,
  and gets links to wedsites that have referance to the query words.

* Web 3 - The user inserts a query medicines side effects" to a search engine (like in web 2),
  and gets links to websites that their content is probably strongly connected to the query and
  maybe adjusted to the user himself.
